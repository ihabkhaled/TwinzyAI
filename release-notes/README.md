# Release Notes — Twinzy

Use this folder to store release notes for player-visible, operator-visible, integration-visible, or contract-visible changes to Twinzy.

- One file per release, copied from [`release-notes-template.md`](./release-notes-template.md); name it by date and slug, e.g. `2026-07-05-engineering-os-migration.md`.
- Release notes are a pre-release gate item (see [`docs/sdlc/release-checklist.md`](../docs/sdlc/release-checklist.md)) and are linked from the request's `25-release-report.md`.
- Player-facing wording follows the product promise: playful style/vibe language, never identity/biometric framing.
