# Agent Product Map

User journey: Landing (/) -> Game (/game): privacy notice + consent + upload -> processing ->
traits + up to 4 vibe matches + disclaimer -> retry/share. Legal: /privacy, /terms, /help.
Framing rules: playful style/vibe only; scores are fit scores, not accuracy; never identity,
biometric, or lookalike claims; disclaimer always visible. Free product — no upsells.
