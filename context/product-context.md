# Product Context

Twinzy: free mobile-first game. Upload one photo -> get playful public style/vibe matches from
written visible traits. Entertainment only; results framed as style/vibe fit (0-100 score,
strong/medium/weak verdict), always with a disclaimer. Never face recognition, identity
matching, or biometrics. Photo never stored. No payments. Routes: / (landing), /game, /privacy,
/terms, /help.
